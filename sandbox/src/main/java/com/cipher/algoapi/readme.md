            参赛者会看见三扇关闭了的门，其中一扇的后面有一辆汽车，选中后面有车的那扇门就可以赢得该汽车，
            而另外两扇门后面什么都没有。当参赛者选定一扇门，但开启的时候，节目主持人会开启剩下两扇门的
            其中一扇，这扇门背后一定没有汽车。主持人会问参赛者要不要换另一扇门。
            问题是：换另一扇门会否增加参赛者获奖概率？
